CloudStack集群部署

CloudStack，Java开发的开源云计算管理平台，用于管理和部署大规模的虚拟化环境，支持 IaaS（基础设施即服务）模型，以其简洁的架构广泛应用于私有云、公共云和混合云场景

集群架构

  • 172.100.100.160 manager
  • 172.100.100.161 cloud01
  • 172.100.100.162 cloud02

2.配置管理节点

2.1 部署MySQL数据库

2.2 创建yum仓库

vi /etc/yum.repos.d/CloudStack.repo 

[cloudstack]
name=cloudstack
baseurl=http://download.cloudstack.org/centos/$releasever/4.18/
enabled=1
gpgcheck=0

2.3 安装管理服务

yum install -y cloudstack-management

2.4 初始化数据库

cloudstack-setup-databases cloud:CloudStack_2025@localhost --deploy-as=root:CloudStack_2025

2.5 启动管理服务

cloudstack-setup-management
