Ansible自动化运维工具生成Linux系统巡检报告

AnsibleHealthCheck，基于Ansible的自动化系统巡检工具，通过Jinja2模板将收集到的服务器关键信息，如操作系统、CPU、内存、磁盘、网络连接等，渲染成详细的HTML巡检报告。此外，巡检报告还支持通过邮件发送，以便相关负责人实时了解整个系统的运行状况

1.安装Ansible、Git

2.下载安装包

cd /etc/ansible
git clone https://github.com/liushiju/ansible-HealthCheck.git
mv ansible-HealthCheck HealthCheck && cd HealthCheck

3.配置巡检组件

3.1 配置过滤器插件

vi ansible.cfg


filter_plugins     = /etc/ansible/HealthCheck/filter_plugins

3.2 配置资源清单

vi hosts


[all]
ops
master
node01
node02
ceph01
ceph02
ceph03

3.3 配置邮箱

cat os-check/defaults/main.yaml 

---
check_day: "{{ '%Y-%m-%d' | strftime }}"
check_report_path: /tmp
check_report_file_suffix: "-{{ check_day }}"

check_mail_host: ""
check_mail_port: ""
check_mail_username: ""
check_mail_password: ""
check_mail_to: []
check_mail_subject: "System Check Report [{{ check_day }}]"

4.执行巡检任务

ansible-playbook -i hosts roles/os-check.yaml

5.配置定时任务

6.验证巡检报告
