Linux系统部署病毒防护平台ClamAV

ClamAV，即Clam AntiVirus，轻量级开源跨平台防病毒引擎，专注于检测病毒、木马、恶意软件及其他安全威胁，提供实时防护、按需扫描与病毒定义库自动更新，广泛用于服务器及终端设备的病毒扫描，属于企业级威胁检测解决方案

1.下载ClamAV

sudo yum -y install clamav clamav-server clamav-data clamav-update clamav-filesystem clamav-scanner-systemd clamav-devel clamav-lib clamav-server-systemd

2.部署私有病毒库

2.1 下载病毒库更新工具

pip3 install cvdupdate

2.2 更新病毒库

cvdupdate update

2.3 配置病毒库定时更新

crontab -l

30 03 * * * /usr/local/bin/cvd update

2.4 启动私有病毒库

nohup cvdupdate serve &

3.配置私有病毒库

sed -i -e "s/^Example/#Example/" /etc/freshclam.conf
sed -i -e "s/^Example/#Example/" /etc/clamd.d/scan.conf
echo "PrivateMirror 私有库ip:8000" >> /etc/freshclam.conf

4.验证病毒库

freshclam

5.启动扫描器

sudo systemctl start clamav-freshclam.service
sudo systemctl enable clamav-freshclam.service
